How to Add a New Product

      
Use this when you have a new product to sell

      
    
        
  1. Go to Stripe Dashboard → Products → Create
    2. 
        
  2. Enter product name, description, and price
    3. 
        
  3. After creating, go to Payment Links → New and select your product
    4. 
        
  4. Copy the buy.stripe.com/... URL
    5. 
        
  5. Tell Claude: "Add [product name] at $[price] with this Stripe link: [URL]"
    6. 
        
  6. Or use the /prml-product skill in Claude Code
    7. 
      

    


    
    

      
How to Update a Price

      
Stripe prices are immutable — you archive old and create new

      
    
        
  1. Go to Stripe Dashboard → Products → find the product
    2. 
        
  2. Click the old price → Archive
    3. 
        
  3. Add a new price at the correct amount
    4. 
        
  4. Create a new Payment Link for the updated price
    5. 
        
  5. Tell Claude: "Update [product] price to $[new price]"
    6. 
        
  6. Claude will update products.json, shop.html, and push to Render
    7. 
      

    


    
    

      
How to Generate Product Photos

      
Using Google Nano Banana Pro for white lightbox shots

      
    
        
  1. Open the photo pipeline doc: 16_CLAUDE_OPS/PRODUCT_PHOTO_PIPELINE.md
    2. 
        
  2. Copy the prompt for your product type (card, keychain, pendant, stand)
    3. 
        
  3. Paste into Nano Banana Pro → generate 3-4 variations
    4. 
        
  4. Pick the best one, crop to 1200x1200px square
    5. 
        
  5. Save as tap-[product]-[color].jpg
    6. 
        
  6. Drop in images/nfc/ and tell Claude to update the catalog
    7.

Royalties & Music Business

      
Collect Every Dollar You Are Owed

      

        

          
What is ASCAP?

          
American Society of Composers, Authors and Publishers. They collect performance royalties when your music is played live, on radio, TV, streaming, or in businesses. You register your songs as "works" and get paid when they are performed. Action: Register at ascap.com â†’ Create account â†’ Add works.

        

        

          
What is SoundExchange?

          
Collects digital performance royalties â€” money from Pandora, SiriusXM, internet radio, and non-interactive streaming. This is separate from ASCAP/BMI. You must register as both an artist AND a rights owner. Action: Register at soundexchange.com.

        

        

          
Performance Royalties from Live Shows

          
Every time you perform live, you can collect royalties IF your setlist is registered with your PRO (ASCAP or BMI). Process: 1) Register songs as works, 2) After each show, submit a setlist to ASCAP/BMI online, 3) They pay you per performance. Beltline shows = royalty opportunity.

        

        

          
Copyright Registration (U.S. Copyright Office)

          
Your music is copyrighted the moment you create it, but registering gives you legal power â€” you can sue for damages, it's public record, and it strengthens licensing deals. Action: copyright.gov â†’ Register a Work â†’ Sound Recording (SR). ~$65 per work or $85 for a group.

        

        

          
The MLC (Mechanical Licensing Collective)

          
Collects mechanical royalties from interactive streaming (Spotify, Apple Music, etc.). This is the money for the composition itself, not the performance. If you wrote the song, you are owed mechanicals. Action: Register at themlc.com.

let uploadedPhotos = [];

function handlePhotoDrop(e){
  e.preventDefault();
  document.getElementById('photo-drop').style.borderColor='rgba(254,1,3,.3)';
  handleFiles(e.dataTransfer.files);
}
function handlePhotoSelect(e){ handleFiles(e.target.files); }

function handleFiles(files){
  Array.from(files).forEach(f=>{
    if(!f.type.startsWith('image/')) return;
    const reader = new FileReader();
    reader.onload = (e)=>{
      uploadedPhotos.push({name:f.name, src:e.target.result, type:f.type});
      renderPhotoGrid();
    };
    reader.readAsDataURL(f);
  });
}

}

function changePw(){
  const np = document.getElementById('new-pw').value;
  const cp = document.getElementById('confirm-pw').value;
  const msg = document.getElementById('pw-msg');
  if(!np){ msg.style.display='block'; msg.style.color='var(--red)'; msg.textContent='Enter a new code.'; return; }
  if(np!==cp){ msg.style.display='block'; msg.style.color='var(--red)'; msg.textContent='Codes do not match.'; return; }

  msg.style.display='block'; msg.style.color='var(--green)'; msg.textContent='Access code updated.';
