Knowledge Base
Tutorials & Walkthroughs
How to Add a New Product
Use this when you have a new product to sell
  1. Go to Stripe Dashboard → Products → Create
  2. Enter product name, description, and price
  3. After creating, go to Payment Links → New and select your product
  4. Copy the buy.stripe.com/... URL
  5. Tell Claude: "Add [product name] at $[price] with this Stripe link: [URL]"
  6. Or use the /prml-product skill in Claude Code
How to Update a Price
Stripe prices are immutable — you archive old and create new
  1. Go to Stripe Dashboard → Products → find the product
  2. Click the old price → Archive
  3. Add a new price at the correct amount
  4. Create a new Payment Link for the updated price
  5. Tell Claude: "Update [product] price to $[new price]"
  6. Claude will update products.json, shop.html, and push to Render
How to Generate Product Photos
Using Google Nano Banana Pro for white lightbox shots
  1. Open the photo pipeline doc: 16_CLAUDE_OPS/PRODUCT_PHOTO_PIPELINE.md
  2. Copy the prompt for your product type (card, keychain, pendant, stand)
  3. Paste into Nano Banana Pro → generate 3-4 variations
  4. Pick the best one, crop to 1200x1200px square
  5. Save as tap-[product]-[color].jpg
  6. Drop in images/nfc/ and tell Claude to update the catalog
How to Push Website Changes
Deploy updates to prmlrecords.com via Render
  1. Open PowerShell and navigate: cd Desktop\PRML_STUDIO\04_WEBSITE\prmlrecords.com\v3_current\prml-site
  2. Stage changes: git add -A
  3. Commit: git commit -m "Description of changes"
  4. Push: git push origin master
  5. Render auto-deploys in ~60 seconds
  6. Or just tell Claude "/deploy" or "push"
How to Add Vercel Environment Variables
For the checkout app at prml-checkout.vercel.app
  1. Open PowerShell, navigate to checkout-app folder
  2. Run: vercel env add VARIABLE_NAME production
  3. Paste the value when prompted
  4. Redeploy: vercel --prod --yes
  5. Currently needed: STRIPE_WEBHOOK_SECRET
