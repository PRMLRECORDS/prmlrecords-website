Daily Update Cheatsheet
To Add A Testimonial
1. Open your Google Sheet
2. Go to the Testimonials tab
3. Add a new row with name, role, text, rating
4. Set approved to TRUE when ready to show
5. Refresh testimonials.html — it auto-updates
To Add A Product
1. Open your Google Sheet
2. Go to the Products tab
3. Add a new row — fill all columns
4. Set category to match the page
5. Set active to TRUE
6. Refresh the page — auto-updates
To Change A Price
1. Open your Google Sheet
2. Find the product in Products tab
3. Edit the price column (numbers only, no $)
4. Save — site updates automatically
To Hide Something
1. Find the row in the right tab
2. Change the active or approved column to FALSE
3. It disappears from the site immediately
